# Minecraft小工具仓库  

AI辅助开发的《我的世界》实用工具集，简化游戏操作流程  

## 工具列表  
- **服务器配置修改器**：快速调整服务器参数（含v1.0基础版与v2.0优化版）  
- **超平坦代码生成器**：自定义地形参数，一键生成超平坦世界代码  


## 使用说明  
### 直接运行（推荐）  
1. 进入 `编译后的exe` 文件夹  
2. 双击对应 `.exe` 文件启动工具  


### 源码运行（开发者）  
1. 安装 Python 3.6+ 环境  
2. 进入 `编译前源代码` 文件夹  
3. 执行 `pip install 依赖名` 安装所需库（依赖需根据代码 `import` 手动识别）  
4. 运行 `python 脚本名.py` 启动工具
